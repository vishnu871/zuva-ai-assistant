# Zuva Life Knowledge Base

This knowledge base powers an AI assistant for the Zuva Life website. It is assembled from publicly available sources (the live website at zuva.life, the Longevity Letters Substack, LinkedIn, Facebook, Instagram, YouTube, Urbanaut, and event listings) and Zuva's own brand materials.

## Source note on accuracy
- Primary source of truth for public-facing facts is the live website: https://www.zuva.life
- Where a fact was not found publicly, this base states "Not publicly found" instead of inventing it.
- Some framework elements (four lenses, six-month identity arc, three narrative pillars) appear in Zuva's internal brand materials but are not published on the public website. These are labelled "Internal framework (not currently published)" so the assistant does not present them as public claims.
- Traction figures (750+ participants, 25+ experiences, funds raised) come from a May 2026 pitch deck and should be re-verified before use in any external answer.
